# 01. Two Sum

## Way 1 ：暴力枚举，两层循环

分析：

### Solution

#### 1.C Sharp

```C#
public class Solution {
    public int[] TwoSum(int[] nums, int target) {
        int [] result = new int[2];
        int size = nums.Length;
        for(int i = 0; i < size - 1; i++){
            for(int j = i + 1; j < size; j++){
                if(nums[i] + nums[j] == target){
                    result[0] = i;
                    result[1] = j;
                    return result;
                }
            }
        }

        return result;
    }
}
```
